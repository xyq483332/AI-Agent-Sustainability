### 2.2 Bug 修复 (已完成)

#### HTTPException 处理问题
- **问题**: `register_plugin` 端点中的 try-except 块捕获了所有异常，包括 HTTPException
- **影响**: HTTPException (如 403) 被重新抛出为 500 状态码
- **解决方案**: 在 except 块中重新抛出 HTTPException
- **验证**: 所有集成测试通过

### 2.3 包结构修复 (已完成)

#### 添加 `src/__init__.py`
- **问题**: `src` 目录缺少 `__init__.py` 文件，无法作为 Python 包导入
- **影响**: 集成测试无法正确导入 API 模块
- **解决方案**: 添加 `src/__init__.py` 文件
- **验证**: 所有集成测试通过
